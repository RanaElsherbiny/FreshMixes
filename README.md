FreshMixes is a modern one-page restaurant website built with HTML, CSS, and JavaScript. It showcases a variety of healthy, freshly made salads designed for every lifestyle — whether you love chicken, beef, seafood, or prefer vegetarian options.

The site highlights FreshMixes’ commitment to freshness and balanced nutrition, featuring vibrant visuals, smooth scrolling sections, and interactive elements that let visitors easily explore the menu, ingredients, and contact information — all on a clean, responsive layout optimized for all devices.
